# 📘 Important File System Commands

# Check disk usage

```bash
df -h
```

---

# Check inode usage

```bash
df -i
```

---

# Check folder size

```bash
du -sh *
```

---

# Find large files

```bash
find / -type f -size +500M
```

---

# Check mounted filesystems

```bash
mount
```

---

# List block devices

```bash
lsblk
```

---

# Check filesystem type

```bash
blkid
```

---

# Disk partitions

```bash
fdisk -l
```
